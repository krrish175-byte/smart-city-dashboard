Smart City Dashboard

by krrish175-byte
A simple and elegant Smart City Dashboard built using HTML, CSS, JavaScript, and Python Flask, integrating OpenWeatherMap APIs to display:
	•	 Real-time weather
	•	 Air quality index
	•	 Location info
	•	 Charts for weather & AQ metrics

All with clean UI and a lightweight backend.

 Tech Stack

Frontend
	•	HTML
	•	CSS
	•	JavaScript
	•	Chart.js

Backend
	•	Python
	•	Flask
	•	Requests
	•	dotenv
